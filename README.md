# glab-preset
> Gitlab preset tool.

## installation
```shell
pip install glab-preset -U
```

## usage
```shell
$ glab-preset --help
# create diffient type
glab-preset --type=reac_oss
glab-preset --type=reac_k8s

Usage: glab-preset [OPTIONS]

Options:
  --type [react_oss]
  --help              Show this message and exit.
```
